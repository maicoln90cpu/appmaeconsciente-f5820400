## 🚀 Quick Start

```bash
npm install
npm run dev
```

Acesse `http://localhost:5173`

## 📁 Estrutura

```
src/           → Frontend React (componentes, hooks, páginas)
supabase/      → Backend (31 edge functions, migrations)
docs/          → Documentação completa
e2e/           → Testes E2E (Playwright)
```

## 🛠️ Stack

**Frontend:** React 18 · TypeScript 5 · Vite 5 · Tailwind CSS · shadcn/ui · TanStack Query · React Router  
**Backend:** PostgreSQL + RLS · Auth · 31 Edge Functions · Storage · Realtime  
**PWA:** Service Worker · IndexedDB · Background Sync · Auto-save  
**Testes:** Vitest · Playwright (20+ specs)  
**Monitoramento:** Sentry · Health Checks · Request Tracing · Spike Alerts
